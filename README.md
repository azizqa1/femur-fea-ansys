# Finite Element Analysis of Human Femur Under Static Load

Static structural FEA study of a human femur using ANSYS Workbench,
analyzing stress distribution and deformation under physiological loading.

# Status
Completed — Academic project, Poznan University of Technology (2024–2025)

# Objective
Identify high-stress regions in the femur under single-leg stance loading —
relevant to fracture risk assessment and orthopedic implant anchorage planning.

# Methodology
- Patient-specific femur geometry from segmented CT data
- Boundary conditions: hip joint load + muscle insertion forces
- Analysis: von Mises stress, principal strain, deformation fields

# Key Results
- Maximum stress concentrated at femoral neck region
- Results consistent with clinical fracture pattern literature
- Deformation within expected physiological range

# Tools Used
ANSYS Workbench  Static structural FEA
CT/DICOM data  Geometry source
